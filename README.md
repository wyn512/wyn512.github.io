# 作品集网站

这是一个静态个人作品集网站，展示了个人的项目和自我介绍。

## 项目结构

```
portfolio-site
├── src
│   ├── index.html        # 网站的主页面，包含基本结构和内容
│   ├── css
│   │   └── style.css    # 样式表文件，定义网站的样式
│   ├── js
│   │   └── main.js      # JavaScript文件，实现网站的交互功能
│   └── assets
│       └── fonts
│           └── README.md # 字体文件的使用说明
└── README.md            # 项目的文档文件
```

## 安装和使用

1. 克隆或下载此项目。
2. 在浏览器中打开 `src/index.html` 文件以查看网站。
3. 根据需要修改样式和内容。
